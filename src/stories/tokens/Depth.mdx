# Depth

Shadow tokens communicate elevation and layer hierarchy. Tailwind v4 provides a structured scale of box shadows and inset shadows.

## Box Shadows

## Usage Guidelines

| Elevation | Shadow | When to use |
|---|---|---|
| Flat | `none` | Inline elements, table rows |
| Raised | `shadow-2xs` – `shadow-xs` | Cards at rest, subtle lift |
| Floating | `shadow-sm` – `shadow-md` | Dropdowns, popovers, tooltips |
| Overlay | `shadow-lg` – `shadow-xl` | Modals, drawers, dialogs |
| Maximum | `shadow-2xl` | Full-screen overlays, sheets |
| Pressed | `inset-shadow-*` | Active buttons, pressed states, inset inputs |
