# Grid & Layout

Breakpoints, container widths, and column grid patterns from Tailwind v4.

## Breakpoints

## Columns

Common grid column layouts. Each cell is a `1fr` column within a CSS grid.

export const colCounts = [1, 2, 3, 4, 6, 12];

export const Cell = ({ span = 1 }) => (
  <div style={{
    gridColumn: `span ${span}`,
    background: 'oklch(0.57 0.21 258.29)',
    opacity: 0.15 + (span * 0.05),
    borderRadius: '3px',
    height: '2.5rem',
    display: 'flex',
    alignItems: 'center',
    justifyContent: 'center',
  }}>
    <span style={{ fontSize: '0.7rem', color: '#1e3a8a', fontWeight: 500 }}>
      {span === 1 ? '1 col' : `${span} col`}
    </span>
  </div>
);

<div style={{ display: 'flex', flexDirection: 'column', gap: '2rem', margin: '1.5rem 0' }}>
  {colCounts.map(cols => (
    <div key={cols}>
      <code style={{ fontSize: '0.8rem', color: '#374151', display: 'block', marginBottom: '0.5rem' }}>
        grid-cols-{cols}
      </code>
      <div style={{ display: 'grid', gridTemplateColumns: `repeat(${cols}, 1fr)`, gap: '0.5rem' }}>
        {Array.from({ length: cols }).map((_, i) => (
          <Cell key={i} span={1} />
        ))}
      </div>
    </div>
  ))}
</div>

## Container Widths

Named container sizes map to `max-width` values used with utility classes like `max-w-sm`, `max-w-lg`, etc.

| Token | Value | Common use |
|---|---|---|
| `--container-xs` | `20rem` (320px) | Narrow cards, mobile popups |
| `--container-sm` | `24rem` (384px) | Compact forms |
| `--container-md` | `28rem` (448px) | Modals, dialogs |
| `--container-lg` | `32rem` (512px) | Article width |
| `--container-xl` | `36rem` (576px) | Wide forms |
| `--container-2xl` | `42rem` (672px) | Readable prose |
| `--container-3xl` | `48rem` (768px) | Tablet content |
| `--container-4xl` | `56rem` (896px) | |
| `--container-5xl` | `64rem` (1024px) | Full desktop layout |
| `--container-6xl` | `72rem` (1152px) | Wide desktop |
| `--container-7xl` | `80rem` (1280px) | Max content width |
