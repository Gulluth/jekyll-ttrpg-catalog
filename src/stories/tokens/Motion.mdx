# Motion

Easing curves, transition defaults, and built-in animation presets from Tailwind v4. Motion should feel purposeful — fast enough to not delay the user, slow enough to communicate cause and effect.

## Duration Presets

Tailwind v4 sets the default transition duration to `150ms`. The full utility scale follows the same progression Tailwind has always used:

| Class | Duration | When to use |
|---|---|---|
| `duration-75` | 75ms | Micro-interactions, tooltips |
| `duration-100` | 100ms | Hover states, focus rings |
| `duration-150` | **150ms** | **Default** — most transitions |
| `duration-200` | 200ms | Dropdowns, small reveals |
| `duration-300` | 300ms | Modals, drawers, panels |
| `duration-500` | 500ms | Page-level transitions |
| `duration-700` | 700ms | Dramatic reveals |

| Token | Value |
|---|---|
| `--default-transition-duration` | `150ms` |
| `--default-transition-timing-function` | `cubic-bezier(0.4, 0, 0.2, 1)` (ease-in-out) |
